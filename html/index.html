How-to Guide

  1. First and foremost, please respect the quota limit. Please limit yourself to about 1000 queries.
  2. Type a question in the search box and hit enter.
    • Question must contain at least 3 words and 10 or more characters.
  3. Hover over nodes to see question titles.
  4. Control-click on nodes to navigate to that question on SO.
  5. Double-click on nodes to search for that question.
  6. Node numbers indicate relevance to search query.
  7. Node colors indicate question communities.
